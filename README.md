PyGeomesh is a tool for generating discretized points from geometry.

现在可用的几何图形有：

- 2D 多边形
- 圆饼
- 球
- Box
- 圆柱

图形关系有：

- 差集

采样方式有：

- 网格采样
- 随机采样

## 使用方式

在仓库中 ipynb 均是样例，可以直接运行，或者在线查看
